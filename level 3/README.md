# Level 3 (Advanced) Tasks Portfolio

## 📈 Task 1: Predictive Modeling (Classification)
* *Objective:* Build a machine learning model to predict customer churn.
* *Key Implementations:* Feature scaling, training Random Forest/Logistic Regression models, and optimizing parameters using Grid Search.
* *Tech Stack:* Python, Scikit-learn, Pandas, Matplotlib.

## 📊 Task 2: Building Dashboards with Power BI
* *Objective:* Create an interactive Stock Performance Dashboard using Power BI to present market volatility trends.
* *Key Implementations:* Calculated Bollinger Bands, 200-day Simple Moving Averages (SMA 200), Drawdown metrics, and set up dynamic gauge ranges.
* *Tech Stack:* Power BI Desktop, DAX Modeling.

## 🧠 Task 3: Natural Language Processing (NLP) - Sentiment Analysis
* *Objective:* Perform sentiment analysis on text data to classify customer reviews or comments as positive, negative, or neutral.
* *Key Implementations:* Tokenization, stopword removal, stemming/lemmatization, and visualizing key themes using word clouds.
* *Tech Stack:* Python, NLTK, TextBlob, Pandas.
