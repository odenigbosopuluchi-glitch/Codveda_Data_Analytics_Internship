# Codveda Data Analytics Level 1

This repository contains data analytics tasks focused on data cleaning, exploratory data analysis (EDA), and data visualization using Python.

## 📁 Repository Structure

* *2) Stock Prices Data Set.csv*: Raw dataset used for stock market behavior analysis.
* *cleaned_house_prediction_data.xls*: Clean dataset exported after the Python cleaning process.
* *Data_Cleaning.html*: Python notebook showing the step-by-step data cleaning pipeline.
* *Exploratory_Data_Analysis.html: Statistical analysis and initial exploration of the **Stock Price dataset*.
* *Data Visualization.html: Charts and visual insights generated for the **House Prediction data*.

# Data Analytics Project Portfolio

This repository contains three foundational data analytics projects covering unsupervised machine learning, forecasting, and predictive modeling.

---

## 📊 Projects Overview

### 1. Clustering Analysis
* *Goal*: Group similar data points using unsupervised machine learning.
* *Method*: Applied K-Means Clustering on a housing dataset.
* *Key Tasks*:
    * Cleaned missing values and handled data scaling.
    * Used the Elbow Method to find the optimal cluster size ($K=3$).
    * Segmented properties into Low-Tier, Mid-Tier, and Luxury market groups.

### 2. Time Series Analysis
* *Goal*: Analyze historical data trends over time to forecast future movements.
* *Method*: Trend analysis, seasonality decomposition, and sequential modeling.
* *Key Tasks*:
    * Cleaned and indexed date-time attributes.
    * Evaluated growth patterns and cyclical variations.
    * Generated forecasts for future time periods.

### 3. Regression Analysis
* *Goal*: Predict continuous outcomes based on independent target features.
* *Method*: Supervised machine learning algorithms (Linear Regression).
* *Key Tasks*:
    * Analyzed feature correlations to prevent multicollinerity.
    * Trained predictive models to estimate specific values.
    * Evaluated model accuracy using metrics like R-squared and MSE.
